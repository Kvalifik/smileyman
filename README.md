# SmileyMan

Easily and infinitely fast. Fetches human text from `.vue` files, generates markdown based on origin tag and writes the result to a Markdown file.

## Usage

Hypothetical folder:

```
folder/
  - bob.vue
  - niels.vue
  - landing.vue
```

Hypothetical folder after running `$ smileyman folder/`:

```
folder/
  - bob.md
  - bob.vue
  - niels.md
  - niels.vue
  - landing.md
  - landing.vue
```
